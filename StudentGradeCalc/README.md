# Task 1: Student Grade Calculator

A simple and interactive Java console application designed to calculate student grades based on marks obtained in various subjects.

## Features
- Input marks obtained (out of 100) for each subject.
- Calculate total marks obtained across all subjects.
- Compute the average percentage.
- Assign grades based on the average percentage:
  - **A+**: 90% and above
  - **A**: 80% to 89%
  - **B**: 70% to 79%
  - **C**: 60% to 69%
  - **D**: 50% to 59%
  - **F**: Below 50%
- Display results: Total Marks, Average Percentage, and Corresponding Grade.

## How to Run
Run the `StudGradeCalc.java` file in any standard Java environment or IDE (like IntelliJ IDEA or Eclipse).
