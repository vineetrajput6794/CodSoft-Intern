# CodSoft - Java Development Internship

Welcome to my repository showcasing Java projects completed during the **CodSoft Java Development Internship**. This collection demonstrates foundational and advanced concepts including Object-Oriented Programming (OOP), database integration via JDBC/MySQL, user inputs, validation, and CLI design.

## Repository Projects

This repository contains the following individual projects:

### 1. [Number Guessing Game](./NumberGame)
A fun command-line game where the user guesses a randomly generated number within a limited number of attempts.
* **Key Skills:** Randomization, user input loops, conditional feedback, scores tracking.

### 2. [Student Grade Calculator](./StudentGradeCalc)
A grade calculation system that takes subject marks, calculates totals and average percentages, and maps them to standard academic grades.
* **Key Skills:** Arithmetic operations, user input collection, grading criteria logic.

### 3. [Student Management System](./StudentManagementSystem)
A full-featured CRUD console application to add, remove, search, and list student information.
* **Key Skills:** OOP design patterns, data persistence with MySQL, JDBC drivers, database connection management.

### 4. [Currency Converter](./CurrencyConverter)
A tool to convert monetary amounts between different selected currencies based on currency exchange rate ratios.
* **Key Skills:** User interfaces, input validation, math operations.

### 5. [ATM Machine Interface](./ATM_Machine_Update)
A simulation of standard banking operations (deposit, withdraw, check balance) using OOP structures connecting an ATM UI to a User Bank Account.
* **Key Skills:** Class relationships, input/overdraft validation, state modification.
