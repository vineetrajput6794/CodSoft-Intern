# Task 1: Number Guessing Game

A fun, interactive command-line number guessing game written in Java.

## Features
- The program generates a random number between 1 and 100.
- The user is prompted to guess the number.
- Feedback is provided if the guess is too high, too low, or correct.
- Limited number of attempts per round (e.g., 5 to 10 attempts).
- Option to play multiple rounds.
- Score tracking based on the number of attempts and rounds won.

## How to Run
Run the `NumberGame.java` file in any standard Java environment or IDE.
