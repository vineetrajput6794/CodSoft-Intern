# Task 5: ATM Machine Interface

An interactive console-based ATM interface in Java that simulates ATM operations connected to a user's bank account.

## Features
- **Withdraw Cash**: Allows user to withdraw a specified amount if sufficient funds exist.
- **Deposit Cash**: Add funds to the bank account.
- **Check Balance**: View current account balance.
- **Object-Oriented Design**: Uses distinct classes representing the ATM machine and the Bank Account for clean code structure.
- **Input Validation**: Ensures valid numeric inputs for transactions and checks for overdrafts.

## How to Run
Run the `Main.java` file in any standard Java environment or IDE.
