# Task 4: Currency Converter

A simple Java application designed to convert amounts between different currencies.

## Features
- **Select Currencies**: Allows the user to choose the base currency and the target currency.
- **Fetch Exchange Rates**: Integrates currency rate conversion logic (hardcoded or fetched).
- **Convert Amount**: Calculates the converted value using the exchange rate and the input amount.
- **Input Validation**: Verifies the input amount is valid.

## How to Run
Run the `currencyConverter.java` file in any standard Java environment or IDE.
